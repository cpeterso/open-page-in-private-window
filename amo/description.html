"Open Page in Private Window" is a Firefox extension that enables you to easily open a copy of the current page in a new Private Window. This is useful for reading articles on websites that limit the number of articles you can read per month. This extension does not clear your browsing history or close the original tab. You can activate this extension from the "Open Page in Private Window" context menu item or an optional toolbar button. You can download the code from GitHub.
